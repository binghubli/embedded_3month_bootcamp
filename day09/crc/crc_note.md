# Day9 CRC 校验原理与查表实现笔记
## 1. CRC基础原理
CRC 循环冗余校验，基于模2除法（异或，无进位无借位）；
核心三要素：生成多项式、初始值、结果异或值、输入/输出反转。

## 2. 两种实现方式
1. 按位计算：逐比特处理，代码简单，速度慢，适合极小数据；
2. 查表法：预计算256字节表，一次处理1字节，速度极快，嵌入式标准方案。

## 3. 常用CRC标准参数
1. CRC8：0x07，Init=0x00，XorOut=0x00，无反转
2. CRC16-Modbus：0xA001，Init=0xFFFF，XorOut=0x0000，输入输出反转
3. CRC32：0xEDB88320，Init=0xFFFFFFFF，XorOut=0xFFFFFFFF，输入输出反转

## 4. Modbus CRC16 重点特性
- 多项式 0x8005，硬件反向 0xA001
- 初始寄存器 0xFFFF
- 每字节高低位反转输入
- 最终结果不额外异或
- 低字节在前，高字节在后传输

## 5. 测试向量验证标准
采用行业公开标准测试数组，对比输出校验值，确保算法无误差。
